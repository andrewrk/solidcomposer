SolidComposer keeps track of your samples so that

  1. Your band members always have the files they need to work on the project.
  2. You can't ever lose the ability to work on your own songs.

To save time, you can upload a .zip file containing all the samples.

Some studios, like FL Studio, support exporting a .zip file which you can directly upload to SolidComposer. This .zip file will include the project file and its sample files, so you would never have to deal with this extra step.

Nobody but you and your band members have access to the samples you upload.
